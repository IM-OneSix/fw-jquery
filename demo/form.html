Form 바이딩

form 서비스

설명: 'form' 서비스 객체를 할당 받아 사용.
에트리뷰트: data-bind-value
함수:
pull 화면으로 부터 변수에 값을 받아 오기, 파라메터: 컨트롤러명
push 변수의 값을 화면으로 출력, 파라메터: 컨트롤러명

push 함수

파라메터: controller-컨트롤러명 (메인콘트롤러인경우 생략가능)

Example
<input type="text" data-bind-value="text" />

// script
k$.main(functoin($vo,$svc) {
	var frm = $svc.get('form');

	$vo.text = 'text';
	frm.push('main');
});
